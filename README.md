vim-cheat-sheet
===============

My very own Vim cheat sheet


Navigation
==========

```j``` Move down
```k``` Move up
```m``` Move right
```h``` Move left

```G``` Go to end of file
```gg``` Go to head of file
```$``` Go to end of line (useful with very large lines)

```ctrl-d``` 1/2 page down
```ctrl-u``` 1/2 page up
